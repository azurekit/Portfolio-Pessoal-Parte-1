## 🚀 Como visualizar o projeto  

1. Habilite o **Smooth Scrolling** nas flags do navegador:  
   - **Chrome:** `chrome://flags/`  
   - **Firefox:** `about:config`  

2. Baixe o repositório e abra o arquivo `index.html` no navegador.
